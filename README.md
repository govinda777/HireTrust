# HireTrust

**Hire-smart:** Gateway SaaS de serviços trustless com gestão automática de SLA via Blockchain. Utilizamos Smart SLAs, escrow e cashback para validar a entrega real através de Proof-of-Service, garantindo transparência total e segurança para as partes.

## 🚀 Visão Geral

O HireTrust é uma plataforma que automatiza a confiança entre contratantes e prestadores de serviços. Através de oráculos e registros imutáveis em blockchain, garantimos que o SLA acordado seja monitorado de forma independente e que compensações (como cashback) sejam executadas automaticamente em caso de descumprimento.

## 🛠️ Tecnologias Chave

*   **Smart SLAs:** Contratos inteligentes que traduzem cláusulas de serviço em código.
*   **Blockchain:** Registro imutável de provas de serviço (Cartório Digital).
*   **Oráculos/TEEs:** Agentes de monitoramento verificáveis.
*   **Escrow Automatizado:** Retenção e liberação programática de pagamentos.

## 👥 Atores

1.  **Plataforma:** Orquestradora do ecossistema.
2.  **Prestador:** Fornecedor focado em performance.
3.  **Assinante:** Cliente com garantia de entrega.
4.  **Oráculos:** Validadores de métricas em tempo real.
